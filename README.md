# Estruturas de Dados e Algoritmos — Curso do Galego

Repositório destinado ao estudo de **Estruturas de Dados e Algoritmos**, com foco em exemplos práticos, exercícios e anotações do curso.

## Objetivo

Organizar materiais de apoio para compreender, implementar e testar conceitos fundamentais de programação, como:

- Vetores e listas (ARRAYS)
- Pilhas e filas (QUEUES and STACKS)
- Listas ligadas (LINKED LISTS)
- Árvores (TREES)
- Grafos (GRAPHS)
- Ordenação e busca (SORTING and SEARCHING)
- Análise de complexidade (TIME and SPACE COMPLEXITY)

## Estrutura

À medida que o projeto evoluir, este repositório poderá conter:

- Exemplos de código
- Exercícios resolvidos
- Explicações teóricas
- Comparações entre algoritmos
- Materiais complementares do curso

## Tecnologias

O projeto utiliza as seguintes tecnologias:

- Linguagem de programação: JavaScript
- Ambiente de execução: Node.js
- Gerenciador de pacotes: npm
- Bibliotecas e frameworks: Jest para testes

## Instalação

1. Clone o repositório para sua máquina local:

```bash
 git clone https://github.com/zMatheus22/Estruturas-de-Dados-e-Algoritmos-Galego.git
```

2. Navegue até o diretório do projeto:

```bash
 cd Estruturas-de-Dados-e-Algoritmos-Galego
```

3. Instale as dependências do projeto:

```bash
 npm install
```

4. Execute os testes para verificar se tudo está funcionando corretamente:

```bash
 npm test
```
